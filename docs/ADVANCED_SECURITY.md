## Общие принципы

1.  **Защита в глубине** - каждая уязвимость имеет несколько уровней защиты
2.  **Никаких исключений** - все правила применяются ко всем без исключения
3.  **Нет конфигурации** - все защиты включены по умолчанию и не могут быть отключены
4.  **Прозрачность** - все защиты работают незаметно для пользователя
5.  **Корректность важнее производительности** - безопасность никогда не жертвуется ради скорости
